# ParaView后处理脚本文件说明

本文件夹包含用于CFD（计算流体动力学）数据可视化和后处理的ParaView脚本。所有脚本均使用ParaView 5.11.2生成，主要用于处理AMReX/BoxLib格式的流体仿真数据。

---

## 🎯 核心后处理脚本

### 1. **data_extraction.py** ⭐ **【推荐】**
- **功能**：AMReX数据提取与导出到CSV
- **处理模式**：支持**单个项目**和**批量项目**处理
- **处理流程**：
  - 读取AMReX/BoxLib格式数据
  - 计算速度矢量（Velocity = ux*iHat + uy*jHat）
  - 计算涡量（Vorticity）
  - 提取矩形区域数据
  - 将单元数据转换为点数据
  - 导出为CSV文件
- **输出格式**：CSV表格，包含Points坐标、速度分量、涡量等
- **配置参数**：
  - `BASE_PROJECT_DIR`：项目基础目录
  - `PROJECT_FOLDERS`：项目列表（支持多个）
  - `AMR_LEVEL`：网格细化级别
  - `DATA_SUBFOLDER`：数据子文件夹路径
- **适用场景**：
  - 从原始AMReX数据生成便于分析的CSV格式
  - 批量处理多个项目
  - 为后续误差计算提供数据源

**使用示例：**
```python
# 单个项目
PROJECT_FOLDERS = ['Cylinder2D_BTDF1']

# 批量项目
PROJECT_FOLDERS = ['Cylinder2D_BTDF1', 'Cylinder2D_BTDF2', 'Cylinder2D_BTDF3']

# 运行
exec(open('data_extraction.py').read())  # ParaView中
python data_extraction.py  # 命令行（如果ParaView支持）
```

---

### 2. **l2_error_calculator.py** ⭐ **【推荐】**
- **功能**：计算L2范式误差（速度场准确性评估）
- **处理模式**：支持**单个文件对**和**批量文件对**比较
- **计算指标**：
  - **L2_ux**：x方向速度L2范式误差
  - **L2_uy**：y方向速度L2范式误差
  - **L2_velocity**：速度矢量L2范式误差
  - **Relative_L2_ux%**：相对误差百分比
  - **Relative_L2_uy%**：相对误差百分比
  - **Relative_L2_velocity%**：速度矢量相对误差
- **数据输入**：来自data_extraction.py生成的CSV文件
- **配置参数**：
  - `REFERENCE_FILE`：参考解文件路径
  - `OUTPUT_DIR`：输出目录
  - `AUTO_FIND_FILES`：是否自动扫描所有CSV
  - `RESULT_OUTPUT_FILE`：结果输出文件名
- **输出**：CSV汇总表，包含所有误差指标
- **适用场景**：
  - 评估数值解与参考解的误差
  - 对比不同参数设置下的精度
  - 生成论文中的误差表格

**使用示例：**
```python
# 单个比较
AUTO_FIND_FILES = False
REFERENCE_FILE = 'reference.csv'
COMPARE_FILES = ['file1.csv']

# 批量比较（自动查找）
AUTO_FIND_FILES = True
REFERENCE_FILE = 'reference.csv'

# 运行
python l2_error_calculator.py
```

## 文件分类总结

### 按优先级分类：
1. **⭐ 必用脚本（新增）**：
   - `data_extraction.py` - 数据提取与导出
   - `l2_error_calculator.py` - L2误差计算
   
2. **学习示例**：`DKT-可用来学习循环加载的写法.py`
3. **单帧可视化**：para3.py, para4.py, car.py
4. **批量处理**：para.py, para2.py, Ga178.py
5. **涡量分析**：display vorticity.py, dispalt vorticity 3d.py, Q-criteria-3D.py
6. **粒子沉降**：settling.py, 3d-settling-moive.py
7. **流线显示**：streamline2.py

### 按流动类型分类：
- **圆柱绕流**：para.py, para2.py, para3.py, para4.py, display vorticity.py, cyl-amr.pvsm
- **球体绕流**：dispalt vorticity 3d.py, Q-criteria-3D.py, sphere_vort.pvsm
- **粒子沉降**：Ga178.py, settling.py, 3d-settling-moive.py, 500particle.pvsm
- **汽车绕流**：car.py, car_low.pvsm

### 按雷诺数分类：
- **Re=100**：para3.py, para4.py
- **Re=185**：Re185_amr_oscillate.pvsm
- **Re=200**：para.py, para2.py, display vorticity.py, dispalt vorticity 3d.py
- **Re=300**：car.py
