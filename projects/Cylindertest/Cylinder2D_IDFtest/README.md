# Cylinder2D_IDFtest Delta-Kernel Notes

This case family compares IBM/IDF interpolation kernels for the 2D cylinder case.
The source case `Cylinder2D_IDFtest` uses the smoothed 3-point kernel by default;
the three sibling directories compile explicit `delta2p`, `delta3p`, and `delta4p`
variants.

## Case Map

| Kernel | Case directory | Kernel selection | Restart checkpoint |
|---|---|---|---|
| `delta3pSmoothed` | `projects/Cylindertest/Cylinder2D_IDFtest` | default `IBM_DELTA_KERNEL=30` | `chk00520000` |
| `delta2p` | `projects/Cylindertest/Cylinder2D_IDFtest_delta2p` | `-DIBM_DELTA_KERNEL=2` | symlink to base `chk00520000` |
| `delta3p` | `projects/Cylindertest/Cylinder2D_IDFtest_delta3p` | `-DIBM_DELTA_KERNEL=3` | symlink to base `chk00520000` |
| `delta4p` | `projects/Cylindertest/Cylinder2D_IDFtest_delta4p` | `-DIBM_DELTA_KERNEL=4` | symlink to base `chk00520000` |

All four runs keep the current `drs = 1.9` and `IB_weight = ds_r * drs`.

## How To Rebuild And Submit

Run commands from the target case directory:

```bash
./scripts/compile.sh
dsub -s ./scripts/submit.sh
```

The current force-field runs restart from `checkpoint.begin_step = 520000` and
write no new checkpoints (`checkpoint.chk_int = 0`). The force-field output is
written only when the Cd convergence branch stops the simulation.

## Steady Cf Outputs

The reusable comparison table is:

```text
projects/Cylindertest/Cylinder2D_IDFtest/delta_kernel_cf_profiles_k12.tsv
```

It is generated by:

```bash
python3 projects/Cylindertest/Cylinder2D_IDFtest/scripts/collect_delta_cf_profiles.py
```

That script preserves per-kernel intermediate files:

- `Cf_steady2_step*_retheta.dat`
- `Cf_steady2_step*_smooth_k12.dat`

It also writes:

```text
projects/Cylindertest/Cylinder2D_IDFtest/delta_kernel_cf_profiles_k12_summary.md
```

The smoothed-kernel Cf profile table intentionally uses the previously processed
`delta3psmooth_data/Cf_steady2_step523000_smooth_k12.dat` baseline. The newer
force-field rerun converged one sampling interval earlier at step 522000.
